## Getting Started: Your First Automation

Start small. Here's a 30-minute setup:

### Step 1: Set Up n8n
```bash
# Using Docker (easiest)
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  n8nio/n8n
```

### Step 2: Connect WhatsApp
Use WhatsApp Business API or services like:
- Twilio WhatsApp API
- 360Dialog
- Unofficial WhatsApp Web API (for testing)

### Step 3: Create Your First Workflow
1. Add Webhook node (receives messages)
2. Add IF node (checks for keywords like "price", "delivery")
3. Add HTTP Request node (sends WhatsApp reply)

### Step 4: Test It
Send a message to your WhatsApp number. Watch the magic happen.

## Common Mistakes to Avoid

**1. Over-automating too soon**
Start with FAQs. Add AI later.

**2. No human fallback**
Always have a way to reach a real person.

**3. Ignoring data privacy**
Store customer data securely. NDPR compliance matters.

**4. Not testing edge cases**
What if someone sends an image? A voice note? Plan for it.
